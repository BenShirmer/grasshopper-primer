### 2.1.1 Element* 
###### User Guide to Element* Tools
![IMAGE](images/Element_Icons_Cover.png)
##Element\* is a mesh geometry plugin for Grasshopper, enabling mesh creation, analysis, transformation, subdivision, and smoothing. Element* provides access to mesh topology data using the Plankton half-edge data structure for polygon meshes. This primer will get you up to speed in working with Element* Mesh tools. Element* expands on traditional methods of accessing mesh data
